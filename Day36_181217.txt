



0. DB 연동

	>> 이클립스에서 DBMS 연동을 위해 해당 프로젝트의 BuildPath 에 오라클 JDBC 경로를 등록한다
	>> 혹은 JRE 라이브러리 경로(일반적으로 C:\Program Files\Java\jre1.8.0_191\lib\ext)에 해당 라이브러리 JAR 파일을 등록하면 된다

	>> DB 커넥션 만들기 1 (public class DBConnection)

		- create class DBConnect, Connection conn;
		- 커넥션 클래스를 생성한다. (java.sql.Connection, java.sql.DriverManager)
		- DB 로그인을 위한 문자열 상수 값들을 지정		: 드라이버, URL, 아이디, 비번
		- 생성자에서 드라이버 로딩			: Class.forName(driver)
		- for get connection object, call static getConnection method from DriverManager, and save that into variable conn
		- create method (Connection getConn() { return conn; })

	>> DB 커넥션 만들기 2 (public class DBTest)

		- 앞서 만든 DBConnection 를 객체화 하여 DBTest 내부 connection 객체를 생성한다 (Connection conn)
		- DB 에 날릴 쿼리문을 작성한다 (String sql)
		- 커넥션 객체(conn)로부터 작성한 sql 을 기반으로 prepareStatement 객체를 얻어온다
			- PreparedStatement ps = conn.prepareStatement(sql);
		- sql 내 ? 가 있다면 그 갯수만큼 ps.setInt(1, 80) 혹은 해당하는 데이터형을 지정하고,
		- 결과를 출력할 ResultSet 객체에 쿼리결과를 저장한다 (일종의 CURSOR)
			- ResultSet rs = ps.executeQuery();
