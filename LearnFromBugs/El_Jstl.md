# EL, JSTL 에러 및 유의사항 모음

## Scope 관련 유의점

- EL 이나 JSTL 이나 특정 변수나 데이터를 활용하려면 특정 Scope 내에 속성으로 저장된 것을 대상으로 해야 한다
- Scope : pageScope (pageContext), requestScope (HttpServletRequest, response), sessionScope (HttpSession), applicationScope
